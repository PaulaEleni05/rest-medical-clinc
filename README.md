# Front-End Development CA2 Medical Clinic Application

A modern, full-stack medical clinic management system built with React and Vite. This application provides a comprehensive platform for managing doctors, patients, appointments, prescriptions, and diagnoses.

## Live Site Hosted on Netlify

Click here -> https://rest-medical-clinic.netlify.app/

## What's in This Repository

This is a complete React-based medical clinic application with the following features:

### Core Functionality
**Full CRUD Operations (Create, Read, Update, Delete) for all entities:**
- **Doctor Management** - Manage doctor profiles, specialisations, and contact information
- **Patient Management** - Manage patient information including personal details
- **Appointment Scheduling** - Schedule, view, and manage appointments between patients and doctors
- **Diagnosis Records** - Record and track patient diagnoses and medical conditions 
- **Prescription Tracking** - Create and manage prescriptions with medication details and dosages

### Design & UI
- **shadcn/ui Components** - Modern, accessible UI components
- **Tailwind CSS** - Utility-first styling
- **Tabler Icons** - Professional icon library
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Landing Page** - Professional hero section with features showcase
- **Dashboard View** - Real-time statistics for logged-in users

### Authentication
- **User Registration** - Create new accounts
- **Login System** - Secure authentication with JWT tokens
- **Protected Routes** - Auth-based access control
